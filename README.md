# 2d-vertical-shooter

Unity 2D로 제작한 세로형(Vertical) 슈팅 게임 프로젝트입니다.

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 2d-vertical-shooter |
| 장르 | 2D Vertical Shooter |
| 엔진 | Unity |
| 설명 | 플레이어가 적을 처치하며 생존하는 세로형 슈팅 게임 |

## 개발환경

| 항목 | 버전/도구 |
|------|-----------|
| Engine | Unity (URP 기반 프로젝트) |
| Language | C# |
| IDE | Visual Studio / Rider |
| Platform | Windows |

## 주요 기능

- 플레이어 이동, 발사, 충돌 처리
- 적 스폰 및 적 탄환 발사 로직
- 파워업 프리팹 기반 공격 패턴 확장
- 점수/게임오버 등 씬 기반 게임 진행 관리

## 조작 방법

| 키 | 동작 |
|----|------|
| 방향키 또는 WASD | 플레이어 이동 |
| Space | 발사 |
